# Supported Devices

FanaBridge includes built-in profiles for the devices listed below. Each profile tells FanaBridge what it can control on your wheel — which LEDs are addressable, whether color is supported, and what display modes are available.

> **Don't see your wheel?** Use the **Wheel Profile Wizard** in the FanaBridge settings tab to generate a profile for any wheel not listed here. See [Adding an unsupported wheel](../README.md#adding-an-unsupported-wheel) for details.
